# ProjectZT
业余时间个人启动的第一个未定名的单机独立游戏，截至2019年9月2日，只开发了任务框架、对话框架、部分道具框架、建造系统、制作系统、地图系统、时间系统，底层类多数利用ScriptableObject实现，还自定义了友好的Inspector面板。道具框架暂时止步，剧情框架做了一半不到就暂停了，考虑到Timeline的使用，正在考虑重建。种田系统开发了一半，这个系统没有美术资源真的进行不下去。
